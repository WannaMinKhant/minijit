Minijit
=======

A naive, educational x86-64 machine code JIT-compiler written in C++ from
scratch.

References
----------

    * Intel assembly manuals:
      https://software.intel.com/en-us/articles/intel-sdm
    * x86 Reference:
      http://ref.x86asm.net/
